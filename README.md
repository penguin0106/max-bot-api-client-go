## Быстрый старт

> Если вы новичок, то можете прочитать [официальную документацию](https://dev.max.ru/), написанную разработчиками Max

### Получение токена
Откройте диалог с [MasterBot](https://max.ru/MasterBot), следуйте инструкциям и создайте нового бота. После создания бота MasterBot отправит вам токен.

### Один инстанс на токен
Клиент библиотека гарантирует, что в рамках одного процесса нельзя создать два экземпляра `Api` с одним и тем же токеном.  
Попытка повторного создания вернёт ошибку `ErrTokenInUse`.

Чтобы освободить токен и завершить работу клиента, вызывайте:

```go
// Закрыть соединения и освободить блокировку токена
_ = api.Close()
```
